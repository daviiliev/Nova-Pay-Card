# 💳 NOVA PAY – Premium Payment Card Landing Page

A modern, premium landing page for a digital + physical payment card. Built with **Vanilla HTML, CSS & JavaScript**.

![Desktop View](screenshots/desktop.png)

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 💳 **3D Card** | Interactive card with hover effect |
| 🌈 **Scroll Gradient** | Background color shifts smoothly on scroll |
| 🟡 **Yellow Theme** | Fresh, vibrant color palette |
| ✨ **Glassmorphism** | Modern blur effects on cards |
| 📱 **Fully Responsive** | Perfect on all devices |
| 🔄 **Scroll-Reveal** | Sections appear with smooth transitions |
| 🎯 **Asymmetric Layout** | Fresh, modern structure |
